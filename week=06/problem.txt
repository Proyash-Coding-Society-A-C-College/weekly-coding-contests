


Print the number of prime numbers that are strictly less than n.

Input Format

 A single integer N.

Constraints

 0 <= n <= 5 * 106

Output Format

 Print the number of prime numbers that are strictly less than n.

Sample Input 0

0

Sample Output 0

0

Sample Input 1

2

Sample Output 1

0

Sample Input 2

10

Sample Output 2

4
